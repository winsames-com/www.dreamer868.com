# 步驟二：建立財報

![建立個人財務報表](https://images.pexels.com/photos/8441778/pexels-photo-8441778.jpeg?auto=compress&cs=tinysrgb&fit=crop&h=627&w=1200)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

有了目標之後，下一步是了解「你現在在哪裡」。

就像看醫生需要健康檢查報告，財務規劃也需要「財務健檢報告」。這份報告會呈現你目前的財務全貌，是後續所有分析的基礎。

---

## 什麼是個人財報

![財報的基本概念](https://images.pexels.com/photos/6863332/pexels-photo-6863332.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

個人財報，是把你的財務狀況系統化整理的文件。

**它包含四個主要部分：**

| 項目 | 內容 |
|------|------|
| 收入 | 所有的收入來源 |
| 支出 | 所有的支出項目 |
| 資產 | 你擁有的東西 |
| 負債 | 你欠的錢 |

**為什麼要整理這些？**

很多人對自己的財務狀況只有「大概」的印象。問他月收入多少、月支出多少，答案常常很模糊。

**但「大概」是不夠的。**不知道確切的數字，就無法做準確的規劃。

## 我們會收集什麼資料

![需要提供的資料](https://images.pexels.com/photos/6694543/pexels-photo-6694543.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

在這個步驟，我們會請你提供以下資料：

**收入相關**
- 薪資單或扣繳憑單
- 其他收入證明（租金、股利等）
- 年度報稅資料

**支出相關**
- 固定支出（房租/房貸、保費、貸款等）
- 信用卡帳單（了解消費模式）
- 大額支出紀錄

**資產相關**
- 銀行存款明細
- 投資帳戶對帳單
- 保險保單
- 不動產權狀
- 其他資產（車、珠寶等）

**負債相關**
- 房貸餘額與條件
- 車貸或其他貸款
- 信用卡循環金額
- 其他借款

不用擔心資料不齊全，我們會告訴你需要什麼、如何取得。

## 財報會呈現什麼

![財報的內容結構](https://images.pexels.com/photos/8441861/pexels-photo-8441861.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

整理完成後，你的財報會包含：

**收支表**
| 項目 | 內容 |
|------|------|
| 月收入 | 各項收入的總和 |
| 月支出 | 固定+變動支出 |
| 月結餘 | 收入-支出 |
| 儲蓄率 | 結餘÷收入 |

**資產負債表**
| 項目 | 內容 |
|------|------|
| 總資產 | 所有資產價值 |
| 總負債 | 所有負債餘額 |
| 淨值 | 資產-負債 |

**保障一覽表**
- 現有的保險保障
- 涵蓋的風險類型
- 保額是否足夠

**資產配置圖**
- 各類資產的比例
- 幣別分布
- 流動性分析

## 這個過程會有什麼發現

![可能發現的問題](https://images.pexels.com/photos/7821671/pexels-photo-7821671.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [RDNE Stock project](https://www.pexels.com/@rdne) on [Pexels](https://www.pexels.com)*

很多人在建立財報的過程中，會有一些「啊！原來如此」的發現：

**「原來我每個月花這麼多」**
- 小額支出累積起來很可觀
- 訂閱服務不知不覺變很多
- 沒有意識到的消費習慣

**「原來我的保險是這樣」**
- 有些保障重複了
- 有些風險完全沒保
- 保費佔收入比例太高或太低

**「原來我的淨值是這樣」**
- 可能比想像中高（資產累積有成）
- 也可能比想像中低（負債影響）
- 終於看清楚自己的位置

**「原來這些細節我都沒注意」**
- 房貸利率其實可以談
- 某些帳戶幾乎沒在用
- 資產配置很不平衡

## 資料保密

![資料安全保障](https://images.pexels.com/photos/8441812/pexels-photo-8441812.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

我們理解，財務資料是非常私密的。

**尊茂的資料保護承諾：**
- 所有資料僅供規劃使用
- 不會分享給任何第三方
- 資料安全儲存
- 你可以隨時要求刪除

你提供的資料，是為了幫你做更好的規劃。我們會以最高標準保護它。
