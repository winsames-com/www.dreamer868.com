# 遺產稅與贈與稅基礎知識

![遺產稅與贈與稅](https://images.pexels.com/photos/5668858/pexels-photo-5668858.jpeg?auto=compress&cs=tinysrgb&fit=crop&h=627&w=1200)
*Photo by [Sora Shimazaki](https://www.pexels.com/@sora-shimazaki) on [Pexels](https://www.pexels.com)*

遺產稅和贈與稅，是傳承規劃中必須了解的兩個稅種。

簡單說：生前給叫做贈與，要繳贈與稅；身後留下叫做遺產，要繳遺產稅。

這兩個稅有密切的關係，做傳承規劃時需要一起考慮。

---

## 遺產稅基本概念

![遺產稅概念](https://images.pexels.com/photos/6863332/pexels-photo-6863332.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

**什麼是遺產稅**

人過世後，留下的財產叫做遺產。遺產超過免稅額的部分，要繳遺產稅。

**計算公式**

```
遺產淨額 = 遺產總額 - 免稅額 - 扣除額
應納稅額 = 遺產淨額 × 稅率 - 累進差額
```

**免稅額和稅率（2024 年）**

| 項目 | 金額/稅率 |
|------|----------|
| 免稅額 | 1,333 萬元 |
| 配偶扣除額 | 553 萬元 |
| 稅率 | 10%-20% |

**稅率表**

| 遺產淨額 | 稅率 |
|---------|------|
| 5,000 萬以下 | 10% |
| 5,000 萬-1 億 | 15% |
| 1 億以上 | 20% |

*稅法可能修改，請以最新規定為準。*

## 贈與稅基本概念

![贈與稅概念](https://images.pexels.com/photos/6802042/pexels-photo-6802042.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

**什麼是贈與稅**

生前把財產無償給別人，叫做贈與。贈與超過免稅額的部分，要繳贈與稅。

**計算公式**

```
贈與淨額 = 贈與總額 - 免稅額
應納稅額 = 贈與淨額 × 稅率 - 累進差額
```

**免稅額和稅率（2024 年）**

| 項目 | 金額/稅率 |
|------|----------|
| 每年免稅額 | 244 萬元 |
| 稅率 | 10%-20% |

**稅率表**

| 贈與淨額 | 稅率 |
|---------|------|
| 2,500 萬以下 | 10% |
| 2,500 萬-5,000 萬 | 15% |
| 5,000 萬以上 | 20% |

**重要**：每年免稅額是「每個贈與人」每年 244 萬，可以給不同的人。

## 兩稅的關係

![兩稅關係](https://images.pexels.com/photos/3184418/pexels-photo-3184418.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

遺產稅和贈與稅密切相關：

**互補機制**
- 死前 2 年內的贈與，要併入遺產計算
- 防止臨終前大量贈與逃稅
- 贈與稅已繳的可以扣抵

**選擇考量**

| 考量 | 贈與 | 遺產 |
|------|------|------|
| 時間點 | 可以控制 | 無法控制 |
| 資產增值 | 移轉後增值歸受贈人 | 增值也算遺產 |
| 免稅額 | 每年都有 | 只有一次 |
| 控制權 | 給出去就失去控制 | 到最後都有控制 |

**所以傳承規劃要綜合考慮兩個稅。**

## 常見的扣除項目

![扣除項目](https://images.pexels.com/photos/3184292/pexels-photo-3184292.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

遺產稅可以扣除的項目：

**身份扣除額**

| 項目 | 金額（2024 年）|
|------|----------------|
| 配偶扣除額 | 553 萬元 |
| 直系血親卑親屬扣除額 | 每人 56 萬元 |
| 父母扣除額 | 每人 138 萬元 |
| 重度以上身心障礙 | 每人 693 萬元 |
| 受撫養之兄弟姊妹、祖父母 | 每人 56 萬元 |

**其他扣除**
- 喪葬費：138 萬元
- 死亡前未償債務：可扣除
- 執行遺囑及管理遺產費用：可扣除

**善用扣除額可以大幅減少稅負。**

## 申報時間和流程

![申報流程](https://images.pexels.com/photos/8441778/pexels-photo-8441778.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

**遺產稅申報**

| 項目 | 內容 |
|------|------|
| 申報期限 | 被繼承人死亡日起 6 個月內 |
| 申報人 | 繼承人 |
| 申報機關 | 被繼承人戶籍所在地國稅局 |
| 可否延期 | 可申請延期 3 個月 |

**贈與稅申報**

| 項目 | 內容 |
|------|------|
| 申報期限 | 贈與日起 30 日內 |
| 申報人 | 贈與人 |
| 申報機關 | 贈與人戶籍所在地國稅局 |

**重要提醒**
- 逾期申報會有罰款和利息
- 需要完稅或免稅證明才能辦理繼承登記
- 建議找專業人士協助

## 常見的誤解

![常見誤解](https://images.pexels.com/photos/3184339/pexels-photo-3184339.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

關於遺產稅和贈與稅的常見誤解：

**誤解一：把錢給小孩就不用繳稅**

錯。贈與超過每年免稅額就要繳稅。

**誤解二：把房子過戶給小孩不用繳稅**

錯。房子過戶給小孩是贈與，可能要繳贈與稅。而且還有土地增值稅、契稅等。

**誤解三：保險金不用繳遺產稅**

不一定。要保人和被保險人是否同一人、受益人是誰，都會影響稅務。

**誤解四：財產放在配偶名下就沒有遺產稅**

錯。夫妻各自的財產都是遺產，只是有配偶扣除額。
