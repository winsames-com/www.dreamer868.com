# 不動產移轉的三種方式（買賣、贈與、繼承）

![不動產移轉方式](https://images.pexels.com/photos/6802042/pexels-photo-6802042.jpeg?auto=compress&cs=tinysrgb&fit=crop&h=627&w=1200)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

想把房子或土地從一個人名下移轉到另一個人名下，有三種主要方式：買賣、贈與、繼承。

這三種方式各有不同的條件、流程、和稅務影響。選擇哪種方式，會直接影響你要付多少稅、要準備什麼文件、以及流程多複雜。

這篇文章介紹這三種方式的基本概念。

---

## 方式一：買賣

![買賣移轉](https://images.pexels.com/photos/5668858/pexels-photo-5668858.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Sora Shimazaki](https://www.pexels.com/@sora-shimazaki) on [Pexels](https://www.pexels.com)*

**什麼是買賣移轉**

一方付錢、一方交房子，雙方同意的價格交易。

**特點**

| 項目 | 說明 |
|------|------|
| 需要金流 | 買方要實際付錢給賣方 |
| 價格 | 依照雙方協議的市場價格 |
| 稅務 | 賣方：土地增值稅、房地合一稅 |
|      | 買方：契稅、印花稅 |

**適用情況**
- 真正的市場交易
- 不相識的人之間
- 投資買賣

**注意事項**
- 親人間買賣，如果價格明顯低於市價，可能被認定為贈與
- 需要有實際的金流紀錄
- 銀行貸款通常會看實際成交價

---

## 方式二：贈與

![贈與移轉](https://images.pexels.com/photos/6863332/pexels-photo-6863332.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

**什麼是贈與移轉**

無償把不動產給別人，不收取對價。

**特點**

| 項目 | 說明 |
|------|------|
| 不需要金流 | 不用付錢 |
| 價值計算 | 依公告現值（通常低於市價）|
| 稅務 | 贈與人：贈與稅（超過免稅額）|
|      | 受贈人：土地增值稅、契稅 |

**適用情況**
- 父母給子女
- 夫妻之間
- 家人之間的財產移轉

**注意事項**
- 每年贈與免稅額 244 萬
- 可以分年贈與，善用免稅額
- 受贈人日後出售，成本以受贈時公告現值計算
- 贈與後就失去控制權

---

## 方式三：繼承

![繼承移轉](https://images.pexels.com/photos/3184418/pexels-photo-3184418.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

**什麼是繼承移轉**

原所有權人過世後，依照繼承規定移轉給繼承人。

**特點**

| 項目 | 說明 |
|------|------|
| 發生時機 | 原所有權人死亡後 |
| 價值計算 | 依公告現值，計入遺產 |
| 稅務 | 遺產稅（併入整體遺產計算）|
|      | 免土地增值稅、免契稅 |

**適用情況**
- 只有在原所有權人過世時才發生
- 不能主動選擇時機

**注意事項**
- 繼承免土地增值稅，是很大的優勢
- 繼承人日後出售，持有期間可以併計被繼承人
- 可能涉及多個繼承人，需要協議

## 三種方式的比較

![方式比較](https://images.pexels.com/photos/3184292/pexels-photo-3184292.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

| 項目 | 買賣 | 贈與 | 繼承 |
|------|------|------|------|
| 需要金流 | 是 | 否 | 否 |
| 可選擇時機 | 是 | 是 | 否（身故才發生）|
| 土地增值稅 | 要繳 | 要繳 | 免繳 |
| 契稅 | 買方繳 | 受贈人繳 | 免繳 |
| 贈與稅 | 無 | 可能要繳 | 無 |
| 遺產稅 | 無 | 無 | 可能要繳 |
| 日後出售成本 | 實際買價 | 公告現值 | 公告現值 |
| 持有期起算 | 取得日 | 受贈日 | 可併計被繼承人 |

**沒有哪種方式一定最好，要根據具體情況選擇。**

## 如何選擇

![如何選擇](https://images.pexels.com/photos/8441778/pexels-photo-8441778.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

選擇移轉方式要考慮：

**稅務成本**
- 計算各種方式的總稅負
- 不只看眼前，也要看日後出售

**時間因素**
- 繼承要等身故
- 贈與可以逐年進行
- 買賣可以立即完成

**控制權**
- 贈與後就失去控制
- 買賣後也失去控制
- 繼承前保有完全控制

**家庭因素**
- 子女的需求和狀況
- 繼承人之間的關係
- 是否需要公平分配

**建議**：找專業人士計算和比較，不要憑感覺決定。
