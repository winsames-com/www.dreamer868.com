# 國際貿易的稅務議題

![國際貿易稅務](https://images.pexels.com/photos/3184418/pexels-photo-3184418.jpeg?auto=compress&cs=tinysrgb&fit=crop&h=627&w=1200)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

國際貿易涉及跨境交易，稅務問題比國內交易複雜許多。

進口有關稅和營業稅，出口有退稅。如果有海外公司，還涉及移轉定價和多國稅務申報。不了解這些，可能會多繳稅或產生風險。

這篇文章介紹國際貿易的主要稅務議題。

---

## 進口的稅務

![進口稅務](https://images.pexels.com/photos/6863332/pexels-photo-6863332.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

進口商品要繳的稅：

**關稅**
- 依商品類別有不同稅率
- 從 0% 到 30% 以上不等
- 稅則號列決定稅率

**營業稅**
- 進口時要繳 5% 營業稅
- 以關稅完稅價格加關稅為稅基
- 可以扣抵銷項稅額

**其他稅費**
- 貨物稅（特定商品）
- 菸酒稅（菸酒類）
- 推廣貿易服務費

**計算公式**
```
關稅 = 完稅價格 × 關稅稅率
營業稅 = (完稅價格 + 關稅) × 5%
```

## 出口的稅務

![出口稅務](https://images.pexels.com/photos/5668858/pexels-photo-5668858.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Sora Shimazaki](https://www.pexels.com/@sora-shimazaki) on [Pexels](https://www.pexels.com)*

出口商品的稅務處理：

**零稅率**
- 出口適用營業稅零稅率
- 不是免稅，是零稅率
- 差別在於可以退進項稅額

**退稅**
- 出口可以申請退還進項營業稅
- 每期營業稅申報時申請
- 或申請提前退稅

**申請退稅需要的文件**
- 出口報單
- 相關進項發票
- 營業稅申報書

**注意事項**
- 要有正確的出口證明
- 進項稅額要有合法憑證
- 退稅有一定的審核時間

## 移轉定價

![移轉定價](https://images.pexels.com/photos/6802042/pexels-photo-6802042.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

跨國企業集團內的交易定價：

**什麼是移轉定價**
- 關係企業之間的交易價格
- 不同於市場上的公平交易
- 各國都有規範

**為什麼要關注**
- 可能被稅局調整補稅
- 移轉定價不當有處罰
- 影響各國的稅負分配

**常見的議題**

| 交易類型 | 移轉定價議題 |
|----------|-------------|
| 商品銷售 | 進貨價格是否合理 |
| 服務費 | 管理費、技術服務費 |
| 權利金 | 商標、專利使用費 |
| 融資 | 關係人借款利率 |

**要準備移轉定價報告，證明價格的合理性。**

## 租稅協定

![租稅協定](https://images.pexels.com/photos/3184292/pexels-photo-3184292.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

國與國之間的租稅協定：

**什麼是租稅協定**
- 國與國簽訂的稅務條約
- 避免雙重課稅
- 規範課稅權的歸屬

**主要效益**
- 降低扣繳稅率
- 避免重複課稅
- 提供爭議解決機制

**台灣的租稅協定**
- 台灣有和多國簽訂
- 包括日本、英國、德國等
- 要了解適用條件

**使用租稅協定**
- 要提出申請
- 需符合資格條件
- 要準備相關文件

## 常見的稅務風險

![稅務風險](https://images.pexels.com/photos/8441778/pexels-photo-8441778.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

國際貿易的常見稅務風險：

**進口面**
- 低報價格被海關補稅
- 稅則號列歸類錯誤
- 原產地申報不實

**出口面**
- 退稅文件不齊
- 假出口真內銷
- 退稅被追回

**跨國面**
- 移轉定價被調整
- 海外所得沒申報
- 常設機構認定

**合規建議**
- 正確申報
- 保存文件
- 專業協助
