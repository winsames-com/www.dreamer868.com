# 信託的三方角色（委託人、受託人、受益人）

![信託的三方角色](https://images.pexels.com/photos/3184418/pexels-photo-3184418.jpeg?auto=compress&cs=tinysrgb&fit=crop&h=627&w=1200)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

信託涉及三個主要角色：委託人、受託人、受益人。

了解這三個角色的關係和權責，是理解信託的基礎。

---

## 委託人

![委託人的角色](https://images.pexels.com/photos/8441812/pexels-photo-8441812.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

委託人是「發起信託的人」。

**委託人做什麼**
- 把財產交出來設立信託
- 決定信託的規則和條件
- 指定受託人和受益人
- 設定信託的目的

**委託人的權利**
- 設計信託契約的內容
- 在契約範圍內修改條款（視契約規定）
- 監督受託人執行
- 保留某些權利（如解除信託，視契約）

**誰可以當委託人**
- 有財產的自然人
- 法人也可以設立信託
- 需要有完全行為能力

**常見情況**：委託人就是想傳承財富的父母、想照顧家人的長輩。

---

## 受託人

![受託人的角色](https://images.pexels.com/photos/5668858/pexels-photo-5668858.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Sora Shimazaki](https://www.pexels.com/@sora-shimazaki) on [Pexels](https://www.pexels.com)*

受託人是「管理信託財產的人」。

**受託人做什麼**
- 接受並管理信託財產
- 依照信託契約執行
- 為受益人的利益行事
- 處理信託相關事務

**受託人的義務**
| 義務 | 說明 |
|------|------|
| 忠實義務 | 只能為受益人的利益行事 |
| 善良管理 | 要用專業標準管理 |
| 分別管理 | 信託財產和自己的財產分開 |
| 不自我交易 | 不能自己買賣信託資產 |
| 報告義務 | 定期向委託人或受益人報告 |

**誰可以當受託人**
- 信託業者（銀行、信託公司）
- 自然人（但較少見）
- 律師或會計師事務所（特定情況）

**最常見的受託人是銀行的信託部門。**

---

## 受益人

![受益人的角色](https://images.pexels.com/photos/4262424/pexels-photo-4262424.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Askar Abayev](https://www.pexels.com/@askar-abayev) on [Pexels](https://www.pexels.com)*

受益人是「享受信託利益的人」。

**受益人的權利**
- 根據信託契約取得信託利益
- 要求受託人報告
- 監督受託人執行
- 在特定情況下可以變更受託人

**受益人的特點**
- 可以是委託人本人
- 可以是第三人（子女、配偶等）
- 可以是多人
- 可以設定順位（如先 A 再 B）
- 可以是尚未出生的人

**受益人可能的安排**
| 類型 | 例子 |
|------|------|
| 自益信託 | 委託人自己是受益人 |
| 他益信託 | 委託人以外的人是受益人 |
| 混合型 | 生前自益、身後他益 |

## 三方關係圖

![三方關係](https://images.pexels.com/photos/3184339/pexels-photo-3184339.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

三方的關係可以這樣理解：

```
委託人 ──交付財產→ 受託人
                    │
            依契約管理
                    │
                    ↓
              受益人（領取利益）
```

**流程是：**
1. 委託人把財產交給受託人
2. 受託人依照契約管理
3. 受益人根據契約領取利益

**重點是：**
- 委託人設定規則
- 受託人執行規則
- 受益人獲得利益

## 同一人可以身兼數職嗎？

![角色重疊的情況](https://images.pexels.com/photos/6802042/pexels-photo-6802042.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

**委託人 = 受益人**（自益信託）
- 可以，很常見
- 例如：退休信託、安養信託

**委託人 = 受託人**
- 有限制
- 可能影響信託的效力
- 一般不建議

**受託人 = 受益人**
- 有限制
- 可能構成利益衝突
- 如果是唯一受益人，信託可能消滅

**最常見的安排是**：委託人和受益人可以是同一人（或有重疊），受託人是獨立的第三方（如銀行）。

## 其他可能的角色

![其他角色](https://images.pexels.com/photos/8441778/pexels-photo-8441778.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

除了三方主角，信託中可能還有：

**信託監察人**
- 監督受託人
- 保護受益人權益
- 在受益人無法自己監督時特別重要

**信託保護人**
- 在離岸信託中常見
- 有特定權力（如撤換受託人）
- 提供額外保護

**顧問**
- 提供專業建議
- 不直接管理財產
- 如投資顧問、法律顧問
