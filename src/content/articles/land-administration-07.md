# 找地政士 vs 自己辦理

![地政士服務選擇](https://images.pexels.com/photos/3184292/pexels-photo-3184292.jpeg?auto=compress&cs=tinysrgb&fit=crop&h=627&w=1200)

不動產過戶，可以自己辦理，也可以找地政士代辦。

很多人想省錢，考慮自己跑流程。但這樣真的划算嗎？

這篇文章比較兩種方式的優缺點，幫助你做決定。

## 地政士做什麼

![地政士職責](https://images.pexels.com/photos/6863332/pexels-photo-6863332.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

地政士是專門處理不動產登記的專業人士：

**主要服務**

| 服務 | 內容 |
|------|------|
| 產權調查 | 確認土地建物狀況 |
| 文件準備 | 準備各種申請書表 |
| 稅務申報 | 土增稅、契稅、贈與稅等 |
| 登記申請 | 向地政事務所申請登記 |
| 流程協調 | 和各機關聯繫 |
| 領取權狀 | 取得新的權狀 |

**地政士的價值**
- 專業知識和經驗
- 節省你的時間
- 降低出錯風險
- 處理複雜狀況

## 自己辦理的流程

![自己辦理](https://images.pexels.com/photos/5668858/pexels-photo-5668858.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Sora Shimazaki](https://www.pexels.com/@sora-shimazaki) on [Pexels](https://www.pexels.com)*

如果自己辦理，需要做的事：

**事前準備**
- 了解整個流程
- 知道需要什麼文件
- 了解各種稅的計算

**申報稅務**
- 自己填寫申報書
- 自己計算稅額
- 自己到稅捐處或國稅局

**申請登記**
- 自己填寫登記申請書
- 自己準備所有文件
- 自己到地政事務所送件
- 自己追蹤進度
- 自己領取權狀

**需要跑的機關**
- 稅捐處（土增稅、契稅）
- 國稅局（贈與稅、遺產稅）
- 地政事務所
- 可能還有其他

## 兩種方式的比較

![方式比較](https://images.pexels.com/photos/6802042/pexels-photo-6802042.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Karolina Kaboompics](https://www.pexels.com/@karolina-grabowska) on [Pexels](https://www.pexels.com)*

| 項目 | 找地政士 | 自己辦理 |
|------|---------|---------|
| 費用 | 要付代辦費 | 只有規費 |
| 時間 | 省時，你只需簽名 | 花費大量時間 |
| 專業 | 有專業把關 | 自己要學 |
| 風險 | 出錯機率低 | 可能出錯 |
| 複雜案件 | 可以處理 | 可能處理不來 |
| 便利 | 高 | 低 |

**費用比較**
- 地政士代辦費：通常數千到數萬元
- 自己辦理：只有政府規費
- 但自己辦理要計算時間成本和風險成本

## 什麼情況適合自己辦

![適合自己辦](https://images.pexels.com/photos/3184418/pexels-photo-3184418.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [fauxels](https://www.pexels.com/@fauxels) on [Pexels](https://www.pexels.com)*

以下情況可能可以考慮自己辦理：

**案件非常簡單**
- 單純的買賣
- 沒有特殊狀況
- 文件齊全

**你有時間**
- 可以請假跑流程
- 不介意多跑幾趟

**你有相關知識**
- 了解流程和規定
- 會填寫各種表格
- 知道如何計算稅額

**金額不大**
- 小案件的話，代辦費佔比較高
- 省的錢比較明顯

**但大多數情況，還是建議找地政士。**

## 什麼情況一定要找地政士

![需要地政士](https://images.pexels.com/photos/8441778/pexels-photo-8441778.jpeg?auto=compress&cs=tinysrgb&h=650&w=940)
*Photo by [Kampus Production](https://www.pexels.com/@kampus) on [Pexels](https://www.pexels.com)*

以下情況強烈建議找地政士：

**繼承案件**
- 文件複雜
- 可能有多個繼承人
- 要處理遺產稅

**贈與案件**
- 涉及贈與稅申報
- 稅務規劃需要專業

**有貸款的不動產**
- 要處理塗銷或設定抵押
- 和銀行的協調

**複雜的產權**
- 共有、設定他項權利
- 有糾紛或爭議

**金額大的案件**
- 風險更高
- 專業的費用相對不高

**時間緊迫**
- 需要快速完成
- 沒時間自己學

**費用**

- 了解包含哪些服務
